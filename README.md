# ConstruccionDeSoftware220261

Este proyecto utiliza **Spring Boot** como framework principal para el desarrollo de aplicaciones Java. Se implementa **MySQL** como base de datos relacional para almacenar datos estructurados y **MongoDB** para manejar datos no estructurados, aprovechando la flexibilidad de ambas tecnologías.

## Tecnologías utilizadas
- **Spring Boot**: Facilita la creación de aplicaciones Java con configuraciones mínimas.
- **MySQL**: Base de datos relacional para datos estructurados.
- **MongoDB**: Base de datos NoSQL para datos no estructurados.

## Instalación
1. Clonar el repositorio.
2. Configurar las credenciales de la base de datos en `application.properties`.
3. Ejecutar la aplicación con `mvn spring-boot:run`.

## Uso
- Acceder a la API en `http://localhost:8080`.
- Consultar la documentación de la API para más detalles sobre los endpoints disponibles.
